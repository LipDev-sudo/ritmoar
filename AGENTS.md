# AGENTS.md
## Objetivo
Este repositório faz parte do portfólio profissional LipDev-sudo.
## Regras de execução
- Nunca alterar a visibilidade do repositório.
- Nunca fazer push direto na main.
- Usar uma branch por objetivo.
- Não misturar correções não relacionadas.
- Preservar a arquitetura existente quando possível.
- Não adicionar dependências sem necessidade comprovada.
- Não substituir identidade visual aprovada.
- Não inventar URLs, resultados de testes ou deploys.
## Validação obrigatória
Executar, quando existirem: instalação limpa, typecheck, lint, testes unitários, testes E2E, build e auditoria de dependências.
## Entrega
Informar branch, commits, arquivos alterados, testes executados, resultado do build, URL de deploy, URL da PR, pendências e autorizações necessárias.
